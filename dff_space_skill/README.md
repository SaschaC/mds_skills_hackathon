# dff-funfact-skill

## Description

**dff-funfact-skill** is a simple service that can give random fun facts.

## Quickstart

```bash
pip install -r requirements.txt
```
Run interactive mode
```bash
python run_interactive.py
```
Run tests
```bash
python run_test.py
```
## Resources

* Execution time: 46 ms
* Starting time: 0.1 sec
* RAM: 45 MB
